...: `mkdir Angolackers_Academy` \
...: `cd Angolackers_Academy` \
...: `git clone https://github.com/Angolackers-Academy/bloGC.git` \
em seguida: \
...: `cd bloGC` \
...: `python3 index.py` \
e por fim abra o seu navegador no endereço: `https://localhost:5000/`
